micro:bit Playground

Un panneau de contrôle BLE basé navigateur pour le BBC micro:bit V2. Capteurs, servos, graphiques en direct, modèles 3D et un GamePad — le tout depuis Chrome, sans aucune installation.

Ouvrir le Playground →

Ce qu'il vous faut

  • Un BBC micro:bit V2 (le V1 marche mais n'a pas de capteur sonore)
  • Un câble USB (pour flasher le firmware une fois) ou des piles
  • Chrome ou Edge bureau — ou Chrome sur Android
  • Bluetooth activé sur votre ordinateur
  • En option : un servo sur P1/P2, des enceintes pour le buzzer, un châssis de robot

Votre première session en 5 étapes

1

Flashez le firmware (une seule fois)

Ouvrez makecode.microbit.org → Nouveau projet → cliquez {} JavaScript → effacez tout → collez le contenu de makecode.ts → cliquez Télécharger → glissez le .hex sur le lecteur MICROBIT. Quand la carte affiche un X, elle diffuse en BLE et elle est prête.

2

Ouvrez l'application

Double-cliquez index.html depuis votre téléchargement — il s'ouvre dans votre navigateur par défaut. Si c'est Chrome ou Edge, tout est prêt. Les nouveaux venus voient une superposition de bienvenue qui explique les bases ; appuyez sur Esc pour la fermer.

3

Choisissez votre thème et votre langue

En-tête → menu rouleau de peinture pour choisir 🌙 Stealth · ⚡ Neon · ☁️ Arctic · 🔥 Blaze. En-tête → menu drapeau pour Anglais · Français · Arabe (avec mise en page RTL complète). Les deux préférences persistent au rechargement.

4

Appairez en Bluetooth

Cliquez le gros bouton 🔗 Connecter au micro:bit. Choisissez votre carte dans le sélecteur Bluetooth du navigateur (nommée BBC micro:bit [xxxxx]). La pastille d'état passe au vert — vous y êtes. Le firmware répond HELLO et la télémétrie commence immédiatement.

5

Essayez chaque onglet

Utilisez 1–8 pour passer d'un onglet à l'autre. Une minute par onglet, c'est un excellent premier tour :

🎛

Contrôles

Dessinez sur la grille LED 5×5, envoyez un cœur, essayez le buzzer

👀

Capteurs

Inclinez la carte, tapez dans les mains, couvrez le capteur de lumière

⚙️

Moteurs

Branchez un servo sur P1, faites glisser le curseur

🎮

GamePad

Les flèches de la croix font défiler sur l'écran LED

📈

Graphique

Cliquez 🎲 Simuler pour des données de démo instantanées

🎲

Carte 3D

Inclinez le micro:bit, regardez le modèle 3D vous imiter

Utiliser le Graphique comme un labo ?

Onglet Graphique → cliquez ⏺ Enregistrer, jouez avec la carte, cliquez ⏹ Stop. Ensuite :

  • ▶ Relecture — rejoue en temps réel sur un graphique vide
  • 💾 Enregistrer la session — télécharge en JSON
  • 📷 PNG / 📄 CSV — exporte le graphique ou les données en tableur
  • 📝 Note — ajoute une annotation horodatée sur le graphique

Envoyez vos propres données depuis MakeCode avec bluetooth.uartWriteLine("GRAPH:Distance:" + d) — le navigateur crée automatiquement une ligne colorée pour chaque étiquette.

Essayer les modèles 3D ?

Onglet 6 → 5 modèles interactifs animés par les données de capteurs en direct :

  • 🎲 micro:bit V2 — LEDs, inclinaison, boutons, broches, logo, teinte selon la température
  • 🚗 Robot Buggy — roues, direction (Servo 1), phares
  • 🦾 Bras robotisé — Servo 1 = base, Servo 2 = élévation, boutons = pince
  • 🎯 Jeu d'équilibre — inclinez la carte pour rouler une balle vers les cibles
  • 🌦️ Station météo — thermomètre, soleil/nuage, girouette, barres sonores

Glissez pour pivoter, molette pour zoomer, activez Rotation Auto pour une démo mains libres.
